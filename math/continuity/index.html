Continuous Functions

Continuity is one of the most seminal concepts in real-valued functions.

Similar to definition of functional limits, we can define continuinty in several equivalent ways:

Let $(X,\sigma)$ and $(X,\rho)$ be metric spaces and let $A \subseteq X$. A function $f : A \to Y$ is continuous at a point $x_0 \in X$ if and only if any of the following conditions hold:

$\epsilon-\delta$ defintion: For every $\epsilon >0$, there is a $\delta > 0$ such that $\sigma(x_0,y) < \delta$ implies $\rho(f(x_0), f(y)) < \epsilon$. Note that unlike the defintion of functional limits, $x_0$ must be in the domain $X$.

The definition above is the generaliztion of continuity for real functions where the metric is the absolute value function.

Sequential defintion: If for all sequences $(x_n) \in A$ such that $(x_n) \to x_0$ then $f(x_n) \to f(x_0)$.

Concise defintion: If $x_0$ is a limit point of $A$ then $\lim_{x \to x_0} f(x) = f(x_0)$. Note that If $x_0$ is an isolated point of $A$ then $\lim_{x \to x_0} f(x)$ is not defined (by defintion of functional limits) and hence this defintion would not be valid.

If $f$ is continuous at every point of $A$, then $f$ is said to be continuous on $E$.

Continuity and Compactness

Continuos functions preserve compactness and connectness:

Theorem: Let $f: A \to Y$ be a continuous function from a compact set $ A \subseteq X$ to metric space $Y$. Then $f(A)$ is compact as well.

Let $f:A \to Y$ be a function from a set $A$ in metric space $(X,\sigma)$ to metric space $(X,\rho)$. Then $f$ is Uniformly Continuous on $A$ if for every $\epsilon > 0$ there exists $\delta > 0$ such that for all $x,y \in A$ in $X$, $\sigma(x,y) < \delta$ then $\rho(f(x),f(x))< \epsilon$

Uniform continuity is a stronger property than normal continuity. The key difference is that when we write the proof of unifrom continuity, we can always choose the value of $\delta$ independent of $y$. That is a single $\delta$ would work simeltounesly for all points $y \in A$.

Theorem: A function that is continuious on a compact set $A$ is uniformly continuous on $K$.

Extreme Value Theorem. If $f:A \to X$ is continuous on a compact set $A \subseteq X$, then $f$ attains an absolute minimum and maximum value. That is, there exists $m,M \in A$ such that $f(m) \leq f(x) \leq f(M)$ for all $x \in A$.

The following canonical theorem states that a continuous real function assues all intermediate values on an interval.

Intermediate Value Theorem. Let $f:[a,b] \to \mathbb{R}$ be a real-valued continuious function. If $L \in \mathbb{R}$ satisfies $f(a) < L < f(b)$ or $f(a) > L > f(b)$, then there exists a point $c \in (a,b)$ such that $f(c) = L$
