Metric and Topological Spaces

Metric spaces are abstract generalization of the real line $\mathbb{R}$ that allows the discussion of convergence, limits, continuity, etc. Topological spaces extends considerably the framework for discussing the same concepts.

Let $X$ be a nonempty set. A function $f:X \times X \to \mathbb{R}$ is called a metric, or distance function. If the metric function $f$ satisfies the following condition, we call $(X,f)$ a metric space:

(i)$f(x,y) \geq 0$.

(ii)$f(x,y) = 0$ if and only if $x=y$.

(iii)$f(x,y) = f(y,x)$ for all $x$ and $y$ in $X$.

(iii $f(x,y) \leq f(x,z) + f(z,y)$ for all $x$ , $y$ , and $z$ in $X$ (Triangle Inequality).

(Example 1) the Euclidean metric $f:\mathbb{R} \times \mathbb{R} \to \mathbb{R}$ defined by $f(a,b) = |a-b| = \sqrt{a-b}$ is an example of a metric function in $\mathbb{R}$. Similarly for the set $\mathbb{R^2}$ we define the euclidean metric by $\sqrt{(b_2 - b_1) + (a_2 - a_1)}$ ( Pythagorasâ€™s Theorem). In general, we can define the metric function $\sqrt{\sum_{i=1}^{\infty}(b_i - a_i)}$ for the set $\mathbb{R^n}$.

(Example 2) Let $C[0,1]$ be the set of continous functions from [0,1] into $\mathbb{R}$ and let $f$ and $g$ be in $C[0,1]$. We can define a metric $d(f,g)$ to measure the area between the functions $f$ and $g$:

$$ d(f,g) = \int_{0}^{1} |f(x)-g(x)|dx$$


If p is a point in $\mathbb{R}$ and $\epsilon>0$, the $\epsilon -neighborhood$ of $p$ is the set $$ V_{\epsilon}(p) = \{x\in \mathbb{R}: |x-p| < \epsilon\}$$

In other words, $V_{\epsilon}(p)$ is an open interval centered at $p$ with radius $\epsilon$.

A set $ A \subseteq \mathbb{R}$ is open if for all points $a \in A$ there exists a $\epsilon$-neighborhood $V_{\epsilon}(a) \subseteq A$

Now we can generalize this notion to metric spaces. Let $(X, d)$ be a metric space. If $\epsilon>0$, we define the open ball of radius $\epsilon$ and center $c$, denoted by $B_{\epsilon}(c)$ as follows: $$ B_{\epsilon}(c) = \{ x \in X : d(c,x) < \epsilon \}$$

If $(X,f)$ is a metric space, a neighborhood of $p$ with radius $r$ is a set that contains some open ball $B_{\epsilon}(p)$ around $p$.

A set $A \subseteq X$ is open if for all $a \in A$ and for any arbitrary small $\epsilon > 0$, there exists an open ball $B_{\epsilon}(a) \subseteq A$. A set without its boundaries is open. This is demonstrated in the open set $A$ and closed set $B$ below. Note that in the set $A$ for every $a \in A$, we can find arbitrary small open ball $B_{\epsilon}(a)\subseteq A$. However in the set $B$ which contains its boundaries, if we choose a point on the boundary, it is impossible to find an open ball (with radius $\epsilon > 0$) contained in $B$.

(Proposition 1) Let $X$ be a metric space. Then

(a) $X$ and the empty set $\varnothing$ are open.

(b) An arbitrary union of open sets is open.

(c) Any finite intersection of open sets is open.

This proposition provides a framework to define the concept of topology: Let $X$ be a nonempty set. A set $\mathcal{T}$ of a collection of subsets of $X$ is called a topology for $X$ if

(i) $X$ and the empty set $\varnothing$ are in $\mathcal{T}$.

(ii) Any union of members of $\mathcal{T}$ is also a member of $\mathcal{T}$.

(iii) Any finite intersection of memebers of $\mathcal{T}$ is also a member of $\mathcal{T}$.

The members of $\mathcal{T}$ are called open sets, and $(X,\mathcal{T})$ is called a topological space.

The complement of an open set is a closed set. That is, $B$ is a closed set in the topological space $(X,\mathcal{T})$ if its complement $A = X \backslash B$ is an open set.

Note: There are sets that are both open and closed. There are also sets are neither open nor closed.

Let $(X,\mathcal{T})$ be a topological space. A neighborhood of a point $x \in X$ is a subset $V$ of $X$ that includes an open set $U \subseteq \mathcal{T}$ containing $x$. In other words, $x \in U \subseteq V$. Note that a neighborhood is not necessiarly open. If a neighborhood is an open set then we call it an open neighborhood.

(Example) The set of integers $\mathbb{Z}$ is closed in $\mathbb{R}$. To see this, we just have to observe that the complement of $\mathbb{Z}$ is union of open sets: $\cup_{n=\infty}^{\infty}(n,n+1)$. Furthemore, the set $\mathbb{Q}$ is neither open or closed in $\mathbb{R}$. To see this, it suffice to observe that there is no $\epsilon-neighborhood$ or open intervals $(a,b)$ in $\mathbb{Q}$.
