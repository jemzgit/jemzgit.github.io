Logarithms

Logarithm is the inverse operation of exponentiation. In fact, $y =\log_b x$ is just another way of writing $ b^y = x$.

Logarithm function $f(x) = \log_b x$ is the inverse function of exponential function $g(x)=b^x$. That is:

$$f^{-1}(g(x)) = g^{-1}(f(x)) = x$$

Domain of logarithm function is $x>0$ and the range is the set of real numbers $\mathbb{R}$.

The base $b$ is a positve number $\neq 1$.

If $b > 1$ then the function is increasing. If $ 0 < b < 1$ the function is decreasing.

$ b = 10$ and $b= e$ are the most common bases in math; where $e \approx 2.71828$ is an irrational constant called Euler number.

if $ b = e \approx 2.71828$ we call $f(x) = \log_e x$ the natural logarithm function and sometimes it is written as $\log x$ or $ln \: x$. Confusingly, some other textbooks use $\log x$ to denote $\log_{10} x$.

.

Alos $lg$ is a convention used by many authors to denote a base-2 logarithm

  • $ \log_b (x_1 x_2) = \log_b x_1 + \log_b x_2 $

  • $ \log_b \dfrac{x_1}{x_2} = \log_b x_1 - \log_b x_2 $

  • $ \log_b x^a = a \: \log_b x $

  • $ b^{\log_b \: x} = x $

  • $ \log_b a = \dfrac{1}{log_a b}$

Change of Base Formula

Most programming languages have built-in functions to calculate only natural logarithm such as Math.log() in javascript. For calculating logarithm with different bases such as $\log_5 3$ we use the change of base formula:

$ \log_b a = \dfrac{\log_d a}{\log_d b} $

Where $d$ is usally chosen to be $e$ or $10$. For example $\log_5 3 = (\log_e 3) / (\log_e 5)$.

Theorems

Theorem 1: If $ x < y $ then $ \log_a x < \log_a y $.
