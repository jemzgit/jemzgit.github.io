Interior, Limit Point, and Closure

Topology is the study of the shapes

Let $(X,d)$ be a metric space. A point $x \in A \subseteq X$ is called an interior point of $A$ if there exists an open ball $B_{\epsilon}(x)$ of radius $\epsilon > 0$ centered at $x$ such that $B_{\epsilon}(x) \subset A$ (or in other words, if there is a neighborhood of $x$ which is contained in $X$).

Generally, the point $p$ is an interior point of a subset $A$ of the topological space $X$ if $p$ is contained in an open subset of $A$ (or equivalently, if there exists a neighborhood $N$ of $p$ such that $N \subset X$). The set of all interior points of $A$ is called the interior of $A$ and is denoted by int(A) or $A^{\circ}$.

We can visualize the interior of a set $A$ in topological space $(X,\mathcal{T})$ as the set of all points of $A$ excluding its boundaries. For example, the interior of the half-open interval $[a,b)$ in the real line is the open interval $(a,b)$.

Let $(X,\mathcal{T})$ be a topological space and $A \subseteq X$. Then the interior of $A$ is the union of all subsets of $A$ that are open in $X$. That is,

$$A^{\circ} = \bigcup_{S \in \mathbb{S}} S$$ Where $\mathbb{S} =\{S \in \mathcal{T}\,: S \subseteq A \}$. It follows from the defintion that the interior is the largest open set in $A$.

$$ A^{\circ} = \{ x \in A : \exists \; B_{\epsilon}(x) \subset A \, , \epsilon > 0 \} $$

A set $A$ is open if every point of $A$ is an interior point of $A$.

A subset $F$ of metric space $(X,f)$ is closed if its complement is open. In the set of real line, every closed interval [a,b] is closed. In order to define closed sets and closure in topological spaces we need to introduce the concept of limit points.

Let $(X,\mathcal{T})$ be a topological space. A point $x \in X$ is a limit point (accumulation point or cluster point) of the set $A \subseteq X$ if each neighborhood of $x$ contains a point of $A$ distinct from $x$. In other words, $(A -\{x\} \cap X \neq \varnothing)$. Intuitively, it says that the limit point $x$ can be approximated by all the points in $A$ in the sense that every neighboorhood of $x$ contains a point of $A$ other than $x$. This is a generalization of the concept of 'limits' in euclidean space.

The same definition holds true for metric spaces: Let $(X,f)$ be a metric space and $A \subseteq X$. We call $x$ a limit point of $A$ if for all $\epsilon>0$ there exists $a \in A - \{x\}$ with $d(x,a) < \epsilon$. In other words, there exists $a \in A$ and $a \neq x$ such that $a \in B(x,\epsilon)$. A point $a \in A$ is an isolated point of $A$ if it is not a limit point of $A$.

A set $C \subseteq \mathbb{R}$ is closed if it contains all its limit points.

If $p \in X$ and $p$ is not a limit point of $X$, then $p$ is an isolated point of $X$.

The closure of $A$ is the union of $A$ and the set of all limit points of $A$. Intuitively, the closure of $A$ is all the point in $A$ or 'near' $A$. We denote the closure of $A$ by $\overline{A}$, $A^{cl}$, or Cl($A$). A point in $\overline{A}$ is called a point of closure. Alternatively, we can define a point of closure of $A$ if every neighborhood of $x$ contains a point of $A$. Note that the definition of a point of closure and limit points are almost identical with the important distinction that every neighborhood of the limit point $x$ must contain a point of $A$ other than $x$ itself.

The closure of $A$ is the smallest closed set containing $A$. In other words, it is the intersection of all the closed sets containing $A$.

Obviously, the interior of a set is an open set and the closure is a closed set. Futhermore:

$$ A^{\circ} \subseteq A \subseteq \overline{A}$$

If $A$ is an open set then $A = A^{\circ}$. If $A$ is a closed set then $A = \overline{A}$.

Theorem. A set $A \subseteq X$ is closed if and only if it contains all its limit points.

A closed set contains its own boundaries.

If $A$ is a set in topological space $X$ then closure $\overline{A}$ is closed. Furthermore, the closure of $A$ is the intersection of all closed sets containing $A$; that is, $\overline{A} = \{A \subseteq K\, : K\; \text{is closed} \}$. In other words, the closure $\overline{A}$ is the smallest closed set containing $A$.

The set $A$ is perfect if $A$ is close and every point of $A$ is a limit point of $A$.

If $\overline{A} = X$ then A is called dense in $X$. In other words, if every point of $X$ is either in $A$ or is a limit point of $A$, then $A$ is dense in $X$. For example, the set of rational numbers $\mathbb{Q}$ is dense in the set of real numbers $\mathbb{R}$.

Suppose $(X,f)$ is a metric space and let $A$ be a subset of $X$ and $a \in A$. Then $a$ is a called a boundry point of $A$ in $X$ if and only if $dist(a,A) = dist(a,A^c) = 0$.
